# CM3D2.RotaryBed.Plugin
　メイドさんのエディット、夜伽シーン、撮影モードなどで、カメラを自動的にぐりぐり動かします。利き手をフリーにして捗りましょう。

## 機能
 * 起動すると、起動時のカメラ位置を基点に、自動的に上下左右に動き回ります。
 * カメラ経路の設定などはほとんどできません。衣装やポーズのエディット後、即使える実用性(?)重視。
 * [CM3D2.BinbolusVR.Plugin](https://github.com/pirolix/CM3D2.BinbolusVR.Plugin)との併用が悪魔的ッ…！

## 使い方
1. COM3D2 での下準備
   * Sybaris2 を導入しておいてください。
   * `Sybaris/UnityInjector` 以下に `CM3D2.RotaryBed.Plugin.dll` をコピーします。
1. CM3D2 での下準備
   * Sybaris を導入しておいてください。
   * `Sybaris/Plugins/UnityInjector` 以下に `CM3D2.RotaryBed.Plugin.dll` をコピーします。
1. メイド編集、撮影モードなどで、`K`キーでオン/オフを切替ます。
   * あらゆるモードで起動できますが、実用的なシーンは限定的です。
